## Installation

To install necessary dependencies run the following commands:

```
npm i inquirer@6.5.0
```

```
npm i jest --save-dev
```

```
npm init -y
```

```
npm install validator

```

## Usage

Download it and run the installation commands thn run the following to start generating team page:

```
node index.js
```
